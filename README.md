# Star Trek Space Combat

Godot 4.5 isometrisches 2.5D Space Combat Spiel mit Federation-, Klingon- und
Romulan-Schiffen. Inspiriert von klassischem Star Trek.

## Features
- Vier-Zonen-Schildsystem mit Bleed-Through
- Reputation + Aggro-basierte AI mit Assist-Mechanik
- Faction-System mit Live-Editor
- Hull-Impact-Dekale, Feuer, Zerstörungs-Explosionen
- Beam- und Bolt-Waffen mit Firing-Arcs

## Setup
- Godot 4.5 stable
- Forward+/Vulkan Renderer
